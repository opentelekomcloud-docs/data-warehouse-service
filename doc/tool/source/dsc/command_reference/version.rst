Version
=======

Function
--------

The **version** command is used to display the version number of the DSC.

Format
------

Linux:

.. code-block::

   ./runDSC.sh --version

Windows:

.. code-block::

   runDSC.bat --version

Parameters
----------

None

Usage Guidelines
----------------

Linux:

.. code-block::

   ./runDSC.sh --version

Windows:

.. code-block::

   runDSC.bat --version

System Response
---------------

.. code-block::

   Version: DSC (Gauss Tools 8.0.1)
