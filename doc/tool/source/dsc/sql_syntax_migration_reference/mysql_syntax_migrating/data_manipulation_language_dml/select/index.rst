SELECT
======

This section describes the migration syntax of MySQL DML. The migration syntax determines how the keywords and features are migrated.

In MySQL, if an SQL statement contains SELECT, INSERT, UPDATE, and REPLACE statements, the SQL query can be migrated to GaussDB(DWS).
