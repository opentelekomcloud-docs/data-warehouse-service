Oracle SQL Configuration
========================

Oracle parameters are used to customize rules for Oracle script migration.

Open the **features-oracle.properties** file in the **config** folder and set parameters in :ref:`Table 1 <en-us_topic_0000001188202590__en-us_topic_0218440495_table15565195515413>` as needed.
