Connecting to a Database
========================

You can connect to the database.

Follow the steps below to connect a database:

#. In the **Object Browser** pane, right-click the selected database name and select **Connect to DB**.

   .. note::

      This operation can be performed only on an inactive database.

   The database is connected.

   The status bar displays the status of the completed operation.

   .. note::

      -  Data Studio allows you to login even if the password has expired with a message informing that some operations may not work as expected when no other database is connected in that connection profile. Refer to :ref:`Password Expiry <en-us_topic_0000001234042179__en-us_topic_0185264975_section56881736153111>` for information to change this behavior.
      -  Refer to :ref:`Cancelling Connection <en-us_topic_0000001233922161__en-us_topic_0185264253_section143932723614>` section to cancel the connection to database.
