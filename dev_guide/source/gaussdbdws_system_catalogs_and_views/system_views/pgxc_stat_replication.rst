PGXC_STAT_REPLICATION
=====================

**PGXC_STAT_REPLICATION** displays the log synchronization status of each node in the cluster. All columns except **node_name** are the same as those in the :ref:`PG_STAT_REPLICATION <dws_04_0767>` view. Only users with system administrator permissions can access this view.
