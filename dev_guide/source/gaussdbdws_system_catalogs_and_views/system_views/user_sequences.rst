USER_SEQUENCES
==============

**USER_SEQUENCES** displays sequence information in the current schema.

============== ===================== =====================
Name           Type                  Description
============== ===================== =====================
sequence_owner character varying(64) Owner of the sequence
sequence_name  character varying(64) Name of the sequence
============== ===================== =====================
