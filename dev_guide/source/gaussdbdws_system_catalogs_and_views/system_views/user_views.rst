USER_VIEWS
==========

**USER_VIEWS** displays information about all views in the current schema.

========= ===================== =================
Column    Type                  Description
========= ===================== =================
owner     character varying(64) Owner of the view
view_name character varying(64) View name
========= ===================== =================
