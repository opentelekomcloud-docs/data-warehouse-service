Querying Snapshot Details
=========================

Function
--------

This API is used to query snapshot details using the snapshot ID.

Calling Method
--------------

For details, see :ref:`Calling APIs <dws_02_0062>`.

URI
---

GET /v1.0/{project_id}/snapshots/{snapshot_id}
