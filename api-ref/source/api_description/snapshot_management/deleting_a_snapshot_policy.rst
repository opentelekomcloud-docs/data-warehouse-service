Deleting a Snapshot Policy
==========================

Function
--------

This API is used to delete a snapshot policy.

Calling Method
--------------

For details, see :ref:`Calling APIs <dws_02_0062>`.

URI
---

DELETE /v1.0/{project_id}/clusters/{cluster_id}/snapshot-policies/{id}
