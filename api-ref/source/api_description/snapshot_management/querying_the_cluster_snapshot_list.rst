Querying the Cluster Snapshot List
==================================

Function
--------

This API is used to query the cluster snapshot list.

Calling Method
--------------

For details, see :ref:`Calling APIs <dws_02_0062>`.

URI
---

GET /v1.0/{project_id}/clusters/{cluster_id}/snapshots
