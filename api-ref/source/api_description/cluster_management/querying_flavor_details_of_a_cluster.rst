Querying Flavor Details of a Cluster
====================================

Function
--------

This API is used to query flavor details of a cluster.

Calling Method
--------------

For details, see :ref:`Calling APIs <dws_02_0062>`.

URI
---

GET /v2/{project_id}/clusters/{cluster_id}/flavor

----------------
