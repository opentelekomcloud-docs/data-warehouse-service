Querying Flavor Information
===========================

Function
--------

This API is used to query flavors supported by GaussDB(DWS).

Calling Method
--------------

For details, see :ref:`Calling APIs <dws_02_0062>`.

URI
---

GET /v2/{project_id}/node-types
