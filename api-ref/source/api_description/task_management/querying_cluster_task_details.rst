Querying Cluster Task Details
=============================

Function
--------

This API is used to query cluster task details.

Calling Method
--------------

For details, see :ref:`Calling APIs <dws_02_0062>`.

URI
---

GET /v1/{project_id}/clusters/{cluster_id}/actions/{action_name}
