Querying the Logical Cluster List
=================================

Function
--------

This API is used to query the logical cluster list.

Calling Method
--------------

For details, see :ref:`Calling APIs <dws_02_0062>`.

URI
---

GET /v2/{project_id}/clusters/{cluster_id}/logical-clusters
