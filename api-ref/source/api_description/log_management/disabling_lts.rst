Disabling LTS
=============

Function
--------

This API is used to disable LTS for a cluster.

Calling Method
--------------

For details, see :ref:`Calling APIs <dws_02_0062>`.

URI
---

POST /v1/{project_id}/clusters/{cluster_id}/lts-logs/disable
